# User Guide

# How to Use the Program

1. Run the program
2. Choose an option from the menu:
   - Add deadlines
   - View deadlines
   - Show study habits
   - Show wellness reminders
   - Add to-do tasks
   - View to-do list

3. Type your input when asked
4. Follow the menu until you exit

# Tips
- Always enter correct dates
- Use the to-do list for daily tasks
