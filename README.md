# Study Planner App

# Description
This is a study planner coded in Python that helps students manage deadlines, tasks, habits, and wellness reminders.

# Features
- Add and view deadlines
- To-do list management
- Study habit reminders
- Wellness reminders such as breaks, sleep, exercise

# How to Run
Run the Python file using any Python editor or terminal

# Purpose
This project helps students improve time management and productivity using a simple program.
